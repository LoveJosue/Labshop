# Description
Labstore est une application e-commerce de vente de produits de laboratoire dont le frontend est développé en Vue.js et déployé sur Vercel, puis le backend en Express.js  et déployé sur Render.

## Fonctionnalités
### Déjà présentes
* Catalogue de produits
* Page de détails du produit
* Ajout au panier
* Gestion du pannier
### À venir très prochainement
* Passation de commande avec courriel de confirmation
* Paiement
* Authentification

## Architecture du projet
Labshop/
│── Labshop/         # Frontend Vue.js
│── Server/          # Backend Express.js
│── vercel.json      # Configuration du déploiement

## Technologies utilisées
### Frontend
* Vue.js 3
* TailwindCSS
* Axios
### Backend
* Node.js
* Express.js
* Mongoose pour MongoDB
### Déploiement
* Vercel -> Frontend
* Render -> API

* Hébergeur DNS : Namecheep
